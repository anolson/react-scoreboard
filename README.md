# React Scoreboard

An implementation of the scoreboard project from the Treehouse [Learn React](https://teamtreehouse.com/tracks/learn-react) track.

### Setup

```
yarn install
```

### Running the server
```
yarn start
```

Now visit: http://localhost:8080/
